Kebijakan Privasi

Terakhir diperbarui: 16 Agustus 2025

Aplikasi Sistem Top Performing Sosmed dikembangkan untuk membantu pengguna menganalisis performa postingan di Instagram berdasarkan data insight yang tersedia melalui Graph API Meta.

1. Informasi yang Kami Kumpulkan

Aplikasi ini dapat mengakses data berikut dari akun Instagram dan Facebook yang telah diotorisasi:

  • Data insight posting (reach, impressions, engagement, dsb.)
  • Informasi dasar halaman atau akun (nama akun, ID akun)
  • Tidak ada data pribadi sensitif seperti kata sandi yang kami kumpulkan

2. Cara Kami Menggunakan Data

Data yang dikumpulkan digunakan untuk:

  • Menampilkan analisis performa posting
  • Menyusun rekomendasi top-performing post
  • Tidak digunakan untuk tujuan komersial lain di luar fungsi aplikasi

3. Penyimpanan dan Keamanan Data

Data hanya disimpan secara sementara pada server untuk keperluan pemrosesan. Kami menggunakan langkah-langkah keamanan standar untuk melindungi data dari akses yang tidak sah.

4. Berbagi Data dengan Pihak Ketiga

Kami tidak menjual, menyewakan, atau membagikan data kepada pihak ketiga kecuali diwajibkan oleh hukum.

5. Hak Anda

Pengguna dapat mencabut akses aplikasi ini ke akun Instagram/Facebook kapan saja melalui pengaturan akun Meta.

6. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Setiap perubahan akan diumumkan di halaman ini.
